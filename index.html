Step-by-step deploy guide
1

Upload backend to GitHub

Go to github.com → New repository → name it ma-support-backend → Public → Create. Upload all files from the ma-backend folder you downloaded.

2

Deploy to Render (free)

Go to render.com → sign up with GitHub → New → Web Service → connect ma-support-backend repo. Set:

Build command: npm install Start command: node server.js Plan: Free

Click Create Web Service. Render gives you a URL like https://ma-support-backend-xxxx.onrender.com. Copy it.

3

Add environment variables in Render

In your Render service → Environment tab → add these 4 variables:

TENANT_ID = (from your IT admin — see Step 4) CLIENT_ID = (from your IT admin — see Step 4) CLIENT_SECRET = (from your IT admin — see Step 4) MAILBOX = support@membershipanywhere.com

Render restarts automatically when you save.

4

Send this message to your IT admin

Copy and send exactly this — they need about 15 minutes:

Hi, I need an Azure App Registration to read the support@membershipanywhere.com shared mailbox automatically for our internal support dashboard. Please do the following in Azure Portal: 1. Go to Azure Active Directory → App registrations → New registration - Name: MA Support Analyzer - Account type: Single tenant - Click Register 2. Go to API permissions → Add a permission - Microsoft Graph → Application permissions - Search and add: Mail.Read - Click "Grant admin consent" 3. Go to Certificates & secrets → New client secret - Description: MA Support Analyzer - Expires: 24 months - Click Add — COPY THE VALUE immediately (you can't see it again) 4. Please send me these 3 values: - Tenant ID (from Overview page) - Application (client) ID (from Overview page) - Client Secret VALUE (from step 3) This only reads the support mailbox — it cannot send emails or access anything else. Thank you!
5

Connect this dashboard to your backend

Once Render is deployed, paste your Render URL in the field above and click Save. Then click Test connection to confirm it's working.

From that moment — clicking Today, Last 7 days, etc. reads your real inbox automatically with zero manual work.
